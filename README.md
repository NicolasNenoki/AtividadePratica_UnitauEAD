# AtividadePratica_UnitauEAD

Atividade prática da matéria de *Desenvolvimento de Sistemas Web*:

### Enunciado da atividade:
- Sabemos que Um algoritmo é um 'caminho' para a solução de um problema e, em geral, existem muitos caminhos que levam a uma solução satisfatória, ou seja, para resolver o mesmo problema pode-se obter vários algoritmos diferentes.
- Com base nos tópicos apresentados na disciplina, elabore um algoritmo que apresente um problema e sua respectiva solução, ou seja, o passo a passo para resolvê-lo.
- A representação pode ser feita por meio de uma sequência simples de passos a serem seguidos. O tema para elaboração do algoritmo é voltado para as rotinas diárias de cada indivíduo. Exemplo: ir para o trabalho, levar o filho à escola, ir ao cinema etc. Um dos critérios para avaliação será a originalidade empregada em cada algoritmo.

#### Sobre o Algoritimo:
Este projeto foi desenvolvido com o objetivo de aplicar conceitos básicos de lógica de programação utilizando a linguagem *Python*.
O programa simula um algoritmo voltado para rotinas diárias de trabalho, permitindo que o usuário escolha qual turno trabalha (manhã, tarde ou noite) e, a partir disso, apresenta um passo a passo organizado para auxiliar na preparação diária.

##### Estrutura do código:
- Estruturas de repetição (while)
- Estruturas condicionais (if, elif, else)
- Tratamento de exceções (try / except)
- Entrada de dados com input()
- Conversão de tipos (int())
